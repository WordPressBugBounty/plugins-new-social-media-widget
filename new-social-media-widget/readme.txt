== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/new-social-media-widget/` folder. Or search for it in your WordPress admin.
2. Activate the plugin.
3. Open **Appearance > Widgets** in your admin dashboard.
4. Drag the widget to your sidebar area.
5. Alternatively, open a page and insert the Gutenberg block.
6. Paste your social profile links and save.

== Frequently Asked Questions ==

= How do I configure my social media icon list? =

Go to **Appearance > Widgets** in your admin panel. Add the widget to a sidebar. You can also edit pages and add the Gutenberg block. Paste your social profile links into the fields.

= Can I use a Gutenberg block to display social icons? =

Yes. Search for the "Social Media Widget" block in the editor. Add it to any page. Customize the social profile links inside the block settings.

= Can I change the style of my social profile links? =

Yes. The settings let you choose custom colors. You can set icon background colors and apply hover effects.

= Which social icons are included? =

The free plugin supports 12 major networks. This includes Facebook, Instagram, YouTube, X, and WhatsApp.

= Do these social profile links open in a new tab? =

Yes. You can select if links open in a new tab or the same window.
