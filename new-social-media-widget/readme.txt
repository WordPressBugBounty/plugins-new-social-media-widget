== Installation ==

1. Upload the plugin files to `/wp-content/plugins/new-social-media-widget/` directory, or install through the Plugins screen.
2. Activate the plugin through the Plugins menu.
3. Go to Appearance > Widgets in your dashboard.
4. Drag the Social Media Icon Widget to your preferred widget area.
5. Add your social profile URLs and customize the appearance.
6. Save and check your site.

== Frequently Asked Questions ==

= How do I add the widget to my sidebar? =

Navigate to Appearance > Widgets in your dashboard. Find "Social Media Icon Widget" in the available widgets list and drag it to your sidebar or any widget area. Then add your profile links and save.

= Can I change the icon colors? =

Yes, the widget includes a color picker that lets you set custom colors for icons, backgrounds, and hover states. You can match the icons to your site's color scheme.

= Which social platforms are supported? =

The free version includes 12 popular platforms like Facebook, X (Twitter), Instagram, YouTube, Pinterest, LinkedIn, and more. The premium version supports 30+ platforms including TikTok, Discord, Telegram, and others.

= Do the links open in a new tab? =

Yes, all social profile links open in a new browser tab by default, keeping visitors on your site.

= Can I add custom CSS styles? =

Absolutely. The widget includes a custom CSS field where you can add your own styling rules for complete control over the appearance.

= Is the widget mobile responsive? =

Yes, the icons automatically adjust to look great on phones, tablets, and desktop screens.

= How do I add hover effects? =

In the widget settings, you'll find hover effect options. The free version includes basic hover color changes, while the premium version offers 60+ animation effects.

= What's the difference between free and premium? =

The free version covers essential needs with 12 icons and basic customization. Premium adds 30+ platforms, 60+ hover effects, 10+ transform effects, and advanced styling options.

= Can I use multiple widgets on one page? =

Yes, you can add the widget to multiple areas like sidebar, footer, and header simultaneously. Each instance can have different settings.

= Does this work with page builders? =

The widget works with any theme that supports standard widget areas. It's compatible with Elementor, Beaver Builder, and other page builders.

= How do I get support? =

For questions or issues, post on the plugin support forum and our team will assist you.
